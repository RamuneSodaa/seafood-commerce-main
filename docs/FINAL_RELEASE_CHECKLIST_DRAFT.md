# 最终上线 Checklist 草案

阶段：Phase 2.23F  
性质：正式上线前最终人工核对草案

## 1. 代码 Baseline

- [ ] 当前代码版本已冻结。
- [ ] 已生成 release candidate baseline。
- [ ] review pack 已归档。
- [ ] 如仍不是 git 仓库，已建立完整备份。
- [ ] 没有未记录的本地临时修改。

## 2. Migration

- [ ] 明确生产库是空库还是已有库。
- [ ] 空库使用 clean baseline。
- [ ] 已有库使用 current-db-to-schema diff。
- [ ] migration SQL 已人工审查。
- [ ] 无危险 SQL：
  - `DROP TABLE`
  - `DROP COLUMN`
  - `TRUNCATE`
  - `DELETE FROM`
- [ ] 准生产或临时库演练通过。
- [ ] 备份和回滚方案已确认。

## 3. Seed

- [ ] production seed 顺序为 stores -> products -> inventory -> marketing -> admin bootstrap。
- [ ] 正式门店已确认。
- [ ] 正式商品、SKU、价格已确认。
- [ ] 正式库存已确认。
- [ ] 正式优惠券规则已确认。
- [ ] 新人券 autoGrant 必查通过。
- [ ] 管理员账号不是本地默认 `admin / <legacy-local-password-removed>`。
- [ ] seed 日志不输出密码、hash、token、secret。

## 4. 顾客端

- [ ] 商品列表正常。
- [ ] 分类图标正常。
- [ ] 搜索正常。
- [ ] SKU 弹窗固定底部。
- [ ] 加入购物车正常。
- [ ] 购物车数量、移除、合计正常。
- [ ] checkout 商品、门店、履约方式正常。
- [ ] 会员价正常。
- [ ] 优惠券单选、全用、不用券正常。
- [ ] 兑换码有效/无效提示正常。
- [ ] 凑单提示正常。
- [ ] 下单成功。
- [ ] 待支付订单可取消。
- [ ] 取消订单释放优惠券。
- [ ] 再来一单正常。
- [ ] 我的优惠券和会员中心正常。

## 5. 后台

- [ ] 管理员登录正常。
- [ ] 后台账号和密码为正式配置。
- [ ] 商品管理可读/可操作。
- [ ] 门店管理可读/可操作。
- [ ] 库存管理可读/可调整。
- [ ] 订单管理可读/可处理。
- [ ] 高风险操作有确认。
- [ ] 库存调整记录操作人和原因。
- [ ] 订单状态变更记录操作人。

## 6. 支付

- [ ] 小程序 AppID 已确认。
- [ ] 微信支付商户号已确认。
- [ ] API v3 key 已安全配置。
- [ ] 商户证书/私钥已安全配置。
- [ ] 回调 HTTPS 域名已配置。
- [ ] 回调 raw body 可用。
- [ ] 回调验签通过。
- [ ] 重复回调幂等。
- [ ] 支付成功订单状态正确。
- [ ] 支付成功优惠券变 USED。
- [ ] 用户取消支付后订单仍待支付。
- [ ] 超时取消释放优惠券。
- [ ] 支付金额以后端订单金额为准。

## 7. 小程序配置

- [ ] 小程序 AppID 正确。
- [ ] request 合法域名正确。
- [ ] 业务域名/隐私协议完成。
- [ ] 服务类目符合实际业务。
- [ ] 体验版可访问。
- [ ] 真机核心流程通过。
- [ ] 提审材料准备完成。

## 8. 经营材料

- [ ] 正式商品资料。
- [ ] 正式商品图片。
- [ ] 门店地址、电话、营业信息。
- [ ] 自提说明。
- [ ] 邮寄发货说明。
- [ ] 售后说明。
- [ ] 隐私协议。
- [ ] 用户协议，如需要。

## 9. 监控 / 日志

- [ ] API 错误日志可查。
- [ ] 支付回调日志脱敏可查。
- [ ] 订单状态日志可查。
- [ ] 库存日志可查。
- [ ] 管理员操作日志可查。
- [ ] 敏感信息不进日志。

## 10. 回滚方案

- [ ] 数据库备份可恢复。
- [ ] API 版本可回退。
- [ ] 小程序体验版/正式版回退路径明确。
- [ ] 支付回调异常处理方案明确。
- [ ] 订单/库存/优惠券状态异常人工处理方案明确。
