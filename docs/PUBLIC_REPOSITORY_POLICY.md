# 公开仓库管理规则

本仓库是 Seafood Commerce 项目的公开源码和开发记录仓库。

## 可以提交

- 应用源代码
- Prisma Schema 和安全的 Migration
- 测试代码
- 脱敏文档
- 环境变量示例
- 不含顾客隐私的演示资料
- 程序运行需要的小型静态资源

## 不得提交

- .env 和 .local
- 微信 AppSecret
- 微信支付密钥及证书
- 数据库真实数据
- 顾客资料
- 管理员真实密码
- SendGrid API Key
- review pack
- 大量原始商品图片和设计源文件

原始商品图片继续保存在开发电脑或后续指定的对象存储中。
