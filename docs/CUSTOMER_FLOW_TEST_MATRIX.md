## 回归重点

- 所有金额以后端 quote/create order 为准。
- 前端不信任用户输入金额。
- 待支付订单锁券合理；取消/超时后必须释放。
- 支付成功后优惠券必须 USED，不能复用。
- 再来一单必须重新校验当前价格、库存、SKU、门店，不复用旧价格。
