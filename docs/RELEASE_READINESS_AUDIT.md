# 绿膳荟小程序上线前收口审计

阶段：Phase 2.22A_release_readiness_audit_plan_only  
性质：audit-only / docs-only  
结论日期：2026-06-14

## 总体结论

当前顾客端已经形成较完整的本地可验收商城闭环：浏览商品、分类搜索、SKU 弹窗、购物车、checkout、会员价、优惠券、兑换码、凑单、下单、待支付订单、取消订单释放券、再来一单、优惠券中心、会员中心均已跑通。

但当前仍不建议直接生产发布。上线前最大阻塞项是正式数据库 migration、正式支付真机验收、生产部署域名/HTTPS/环境变量、小程序发布配置、后台运营能力与版本基线。

## 当前不建议继续做的事

- 不建议继续堆新营销功能，先做上线准备。
- 不建议引入现金返利、提现、多级分销。
- 不建议把本地 `db push` 状态直接当生产迁移。
- 不建议在正式支付未验收前宣传“支付已完成上线”。

3. Phase 2.23B：支付准生产真机验收。
